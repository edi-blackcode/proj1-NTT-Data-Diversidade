Utilizando o inspetor de elementos no Chrome

O inspector de elementos é um recurso que nos permite conhecer e entender melhor a estrutura de um site.

Existem três maneiras de inspecionar elemento no Chrome:

  • Clique com o botão direito em um item da página e escolha Inspecionar;
  • Clique no menu de três pontos no canto superior direito da janela e escolha Mais ferramentas > Ferramenta de desenvolvedor;
  • Pressione Ctrl + Shift + C no teclado (Cmd + Option + C no Mac).

Estrutura HTML

A estrutura básica de uma página HTML é composta pelos elementos:

<html> <head> </head> <body> <body> </htm>

Depois que avançamos nos estudos aprendemos a implementar mais elementos HTML, de acordo com a necessidade do projeto:

<head> <meta charset="utf-8"> <title>Título</title> <link rel="stylesheet" href="styles.css"> </head>

Falando sobre tags e seus atributos

Os marcadores de elementos ficam entre tags, ou seja, aquele sinalzinho ali no seu teclado que simbolizam o "menor que" e "menor que", < e >.

Os atributos de tags são utilizados para identificá-los e chamá-los em hiperlinks internos e extenos e no css para a estilização.

Alguns dos principais atributos são:

  • alt: utilizado como descrição nas imagens para o usuário ler sobre a imagem quando o código quebra e a imagem some;
  • id: é o identificador único para aquela tag, e não será usado novamente no código;
  • class: é o identificador em classe, usado várias vezes para identificar vários elementos e chamá-los ao mesmo;
  • href:usado para informar a URL para onde o link levará.

Listas ordenadas e não ordenadas

As listas são muito utilizadas em HTML. As Listas ordenadas (or) e as listas não ordenada (ul - underordened lis) Abaixo está a forma como a colocamos no html e a forma como ela se apresenta na tela do browser:

<ul> <li>Coffee</li> <li>Tea <li>Black tea</li> <li>Green tea <ul>

  • Coffee
  • Tea
  • Black tea
  • Green tea

      <ol> <li>China</li> <li>Africa</li> <li>Brasil</li> <li>Alemanha</li> </ol>

      1. China
      2. Africa
      3. Brasil
      4. Alemanha

      Links internos e externos

      Os links são usados no html para levar o usuário para outra parte do site, podendo ser um link interno que leva para um local dentro do mesmo projeto, em outra mesma página, e podendo ser um link que leva para outro site, externo ao site atual, temos ainda a ancoragem, que nos leva para um tópico específico dentro do conteúdo da página.

      Um ótimo exemplo de ancoragem é a opção back to top localizada abaixo de cada seção aqui na página, elas funcionam como um botão, ao clicar você será direcionado para o topo da página, onde tem um menu com os nomes das seções e se você clicar no título será igualmente direcionado para o conteúdo.

      Um ótimo exemplo de link interno é a opção Blog ==> Blog, que te levará para outra página desse mesmo projeto mas que está em arquivo separado.

      Um ótimo exemplo de link externo é a opção aqui ==> LinkedIn, que te levará direto para meu LinkedIn.
